# 唐老鸭抢红包游戏 v0.17

## 【这个项目是啥？】
一个超级有趣的小游戏！你将控制唐老鸭在屏幕上飞来飞去，抢夺从天而降的红包，积累财富！

## 【它能做什么？】
- **抢红包玩法** - 控制唐老鸭上下左右移动，收集红包赚取金币
- **技能系统** - 三只小鸭子各有特殊技能，游戏开始时随机点名
- **装扮系统** - 给唐老鸭换上不同风格的服装
- **装扮商店** - 花费金币购买新的装扮套装（新功能！）
- **代码统计** - 查看项目代码量的柱状图、饼图和详细数据（新功能！）
- **AI对话** - 与唐老鸭AI助手聊天互动
- **数据统计** - 查看游戏数据和技能使用统计

## 【怎么使用它？】

### 启动游戏
1. 确保已安装 Java 8 或更高版本
2. 确保 MySQL 数据库正在运行
3. 运行数据库初始化脚本：`./init_database.sh`
4. 编译并运行：
   ```bash
   javac -d . src/main/java/Game0_17/*.java
   java Game0_17.Main
   ```

### 游戏操作
- **方向键** - 控制唐老鸭移动
- **Z键** - 让小鸭子说话
- **BEGIN按钮** - 开始游戏
- **FUNCTION按钮** - 打开功能菜单

### 装扮商店使用指南
1. 点击 FUNCTION → 换装系统
2. 在换装界面右下角点击"🛍️ 进入商店"
3. 查看可购买的套装（每套100元）
4. 点击购买按钮购买喜欢的套装
5. 购买成功后返回换装界面即可使用

### 代码统计功能
1. 点击 FUNCTION → 代码统计
2. 选择要统计的代码文件夹
3. **有奖竞猜** - 扫描完成后会弹出竞猜窗口
   - 猜测哪种编程语言的代码最多
   - 猜对了会触发红包雨奖励！
   - 可以选择跳过竞猜
4. 查看五个标签页：
   - **总览** - 关键统计数据卡片
   - **柱状图** - 各编程语言代码量对比
   - **饼图** - 各语言占比分布
   - **详细数据** - 完整的表格统计
   - **代码质量** - 分析和建议
5. 点击"返回游戏"按钮返回主界面

## 【常见问题解答】

**Q: 为什么我不能使用某些装扮？**
A: 除了默认装扮，其他装扮都需要先在商店购买才能使用。

**Q: 购买的装扮是永久的吗？**
A: 是的！一旦购买，装扮将永久保存在数据库中。

**Q: 金币不够怎么办？**
A: 多玩几局游戏，积累更多金币后再去商店购买。
